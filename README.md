# Evaluación 1 – Expansión y Síntesis de Testing Avanzado

Autor: Dylan Nixon Moreno Chávez

## Descripción

Proyecto desarrollado para la asignatura Diseño de Pruebas, Control de Calidad y Mantenimiento.

El sistema integra diferentes técnicas avanzadas de testing utilizando JavaScript, Node.js y Jest.

## Componentes implementados

### Framework híbrido
- Inspirado en Jasmine.
- Pruebas de integración.
- Espías personalizados.
- Generación automática basada en tipos.

### Extensión del algoritmo de búsqueda binaria
- Property-Based Testing.
- Contract Testing.
- Mutation Testing.

### Métricas avanzadas
- Complejidad ciclomática.
- Detección de flaky tests.
- Tiempo de ejecución.
- Relación cobertura/defectos.

### Orquestación de pruebas combinatorias
- Generación automática de casos.
- Priorización por riesgo.
- Aprendizaje basado en ejecuciones previas.

### Pipeline integral
- ESLint.
- Jest.
- Cobertura.
- Mutation Testing.
- ACTS.
- TestCraft.
- Modelo predictivo.

### Investigación e innovación
- Comparación TestCraft vs Jasmine vs Selenium.
- Modelo predictivo de confiabilidad.

## Tecnologías utilizadas

- JavaScript
- Node.js
- Jest
- ESLint
- HTML
